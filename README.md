AssetTrak
An application that allows staff members to scan and checkout assets from the warehouse while providing the business with key insights into asset allocation, whereabouts and utilization.

Features:
User authentication 
Database management
Form validation

Live Demo: 
https://asset-trak.vercel.app/

Tech used:
Next.js
TypeScript
TailwindCSS
Postgres SQL
Prisma
React hook form
Zod
